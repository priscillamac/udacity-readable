## Getting Started

To get started developing right away:

* Install and start the API server
    - `cd api-server`
    - `npm install`
    - `node server`

* To launch the app, open a new terminal window and do the following commands
    - `cd frontend`
    - `npm install`
    - `npm start`

The app will launch on localhost:3000
